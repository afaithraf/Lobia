#Utility

Sizing

Almost everything in Lobia rely on rem and em for it's size, so by altering font-size of the element will affect it's width and height.
You can use the following classes directly on your element to change it's font-size.

xs sm md lg xl xxl 3xl 4xl 5xl 6xl 7xl

Width & Height

w-* for width

h-* for height

Replace * with:

xs sm md lg xl xxl 3xl 4xl 5xl 6xl 7xl auto 100vh 100vw 0 25 (25%) 50 (50%) 75 (75%) 100 (100%)



Margin & Padding (* same as width & height)

margin

.m-* => margin

.mx-* => margin-left, margin-right

.my-* => margin-top, margin-bottom

.mt-* => margin-top

.mb-* => margin-bottom

.ml-* => margin-left

.mr-* => margin-right

padding

.p-* => padding

.px-* => padding-left, padding-right

.py-* => padding-top, padding-bottom

.pt-* => padding-top

.pb-* => padding-bottom

.pl-* => padding-left

.pr-* => padding-right



Border

.b => border

.bt => border-top

.bb => border-bottom

.bl => border-left

.br => border-right

To change border color

.b-(color)



Border Radius

You can change border radius with the default lobia radius

.r => border-radius

.rtl => border-top-left-radius

.rtr => border-top-right-radius

.rbl => border-bottom-left-radius

.rbr => border-bottom-right-radius
